# UI Skill

Purpose:
- Define modern, polished UI design for Rocky’s frontend experience
- Provide guidance for sleek layout, motion, and visual systems
- Keep the experience fresh, expressive, and modern as hell

When to use:
- designing web interfaces for the Rocky assistant
- defining component layout, spacing, and responsive behavior
- selecting typography, color, and motion treatments
- shaping interactions for chat, navigation, and animated assistant states

Design principles:
- minimal, bold visual hierarchy
- fluid motion and expressive transitions
- high-contrast, accessible color and typography
- smart spacing with generous padding and responsive scale
- crisp micro-interactions for hover, focus, and loading states
- modern glassmorphism / neomorphic accents when appropriate
- dynamic dark/light support with vivid accent tones

Implementation guidance:
- use clean component structure with reusable design tokens
- prefer large, readable typography and strong call-to-action styling
- include subtle depth via shadows, gradients, and soft surfaces
- use motion to guide attention, not distract users
- ensure fast, responsive UI across mobile, tablet, and desktop
- make the assistant feel alive through animated entry, hover, and typing states

Tone:
- confident, modern, and polished
- bold but not cluttered
- playful where appropriate, professional overall
- future-forward, tech-savvy, and refined
