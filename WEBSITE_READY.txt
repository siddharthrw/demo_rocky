🛠️ CUSTOMIZATION

Add More Pages:
  1. Add new div in website.html:
     <div id="products" class="page">...</div>
  
  2. Add nav link:
     <a class="nav-link" data-page="products">Products</a>
  
  Done! Rocky will be visible on new page too.

Change Colors:
  Edit CSS variables at top of website.html:
  --primary: #667eea
  --secondary: #764ba2
  --accent: #7ecfb3

Adjust Rocky Speed:
  rockyX += 2  // Change to 1 (slower) or 3 (faster)
