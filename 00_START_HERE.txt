🎯 WHAT YOUR DEMO DOES

User Flow:
  1. User opens http://localhost:5000
  2. Sees beautiful landing page
  3. Rocky sprite is walking/bobbing at bottom-right
  4. User clicks Rocky
  5. Chat modal slides up from bottom
  6. User types a message (e.g., "Who are you?")
  7. Clicks send button (or presses Enter)
  8. Message appears in chat bubble
  9. Flask backend calls Gemini API
  10. Rocky's response appears in chat
  11. User can continue chatting

💡 OPTIMIZATION TIPS

To reduce tokens further:
  1. Reduce maxOutputTokens from 150 → 100
  2. Add local FAQ cache (0-token responses)
  3. Add response caching per user
  4. Implement smart intent detection

To save cost:
  1. Use gemini-2.0-flash-lite (cheaper)
  2. Cache common responses
  3. Batch multiple messages
  4. Use local Llama 2 for simple questions

🐛 QUICK TROUBLESHOOTING

Problem: "ModuleNotFoundError: flask"
Solution: pip install flask flask-cors

Problem: "Address already in use"
Solution: Change port in web_app.py (line 96)

Problem: Rocky not responding
Solution: Check browser console (F12), verify API key in .env

Problem: Slow responses
Solution: Check internet, verify Gemini API is working
